# Installation

## Prerequisites

Make sure you have Node.js 18+ installed on your machine.

## Quick Start

```bash
npx create-next-app@latest my-app --typescript --tailwind --eslint
cd my-app
npm run dev
```

## Manual Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/next-tailwind-ts-starter.git
cd next-tailwind-ts-starter
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```
